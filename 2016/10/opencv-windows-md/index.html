Windows 10, Visual Studio 2015环境下OpenCV 3.1配置和入门

发表于 2016-10-28

OpenCV是计算机视觉的事实标准库，学习OpenCV非常重要，在目前的国情下，在Windows上完成OpenCV的开发和教学工作是非常重要的，还不能指望所有人都有Mac计算机，即使Linux也是不现实的。

我们来详细探讨如何在Windows 10，Visual Studio 2015的环境下来配置OpenCV的学习环境。

下载OpenCV

没啥说的，OpenCV官网上下载，我们要下载最新的3.1版本，所以要点击DOWNLOADS链接，不要下载主页上面的2.4.13版本。在下载页面中下载OpenCV for Windows 3.1即可。

下载下来的是一个opencv-3.1.0.exe文件，其实就是一个自解压的压缩包，可以直接执行也可以7-zip解压，随你便。

最重要的两个目录是build\include和build\x64\vc14\lib目录，这是Visual Studio 2015 C++中要配置的两个重要目录。

如何配置

在Visual Studio 2015中新建一个Visual C++的Empty Project，项目名字和位置自己定义好。

在project上右击选择Properties，在VC++ Directories中配置Include Directories和Library Directories。

要注意属性的配置文件，非常重要。我们配置Debug和x64。是的，OpenCV 3.1已经不支持x86的开发了。

配置完路径后，要在Linker段中的Input中的Additional Dependencies中将OpenCV的opencv_world310d.lib引入。

关闭配置框后一个重要的事情是在顶部工具条中要修改当前的配置。

新建一个C++文件。我们来完成第一个tutorial，Load and Dispaly an Image。

为了学习方便，我们在一个项目中通过菜单和函数来完成不同教程。

编译运行，会收到错误。

这是因为找不到动态链接库。你有两种方法，第一种是将opencv\build\x64\bin\opencv_world310d.dll复制到c:\windows\system目录下，第二种方法是将opencv\build\x64\bin\目录添加到path环境变量中。

或者我们可以把opencv_world310d.dll直接复制到我们生成的.exe目录中。

OK，大功告成！
